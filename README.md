## Quick Start

```bash
# Clone the repository
git clone https://github.com/crsandeep/simple-react-full-stack<img width="1439" alt="Screen Shot 2022-09-11 at 9 27 28 PM" src="https://user-images.githubusercontent.com/40225095/189558940-86d6522e-b40d-42a6-8afd-6db182ae4eb0.png">


# Go inside the directory
cd simple-react-full-stack

# Install dependencies
yarn (or npm install)

# Start development server
yarn dev (or npm run dev)

# Build for production
yarn build (or npm run build)

# Start production server
yarn start (or npm start)
```
